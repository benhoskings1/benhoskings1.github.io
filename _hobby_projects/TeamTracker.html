Project Description

This Coventry Godiva Analytics Solution is a full stack development project that aims to connect the excellent coaching staff at the club with all of the data that they need to take thier coaching to the next level!

One of the primary goals of the project is for the full system to be essentially zero cost!

This has been achieved by purposely selecting free-to-use systems

About the project

The development stack has 4 main sections of operation

  1. Data Input/Acquisition
  2. Data Storage
  3. Data Processing
  4. Data Visualisation

Data Input

Data is obtained using the Strava API.
